# Hero Video

Place your hero background video in this folder.

## Supported Formats
- MP4 (recommended for best browser compatibility)
- WebM (for modern browsers)

## Recommended Specifications
- Resolution: 1920x1080 (Full HD) or higher
- Duration: 10-30 seconds (looping video)
- File size: Under 5MB for optimal performance
- Aspect ratio: 16:9
- Codec: H.264 for MP4

## File Name
Name your video file: `hero-background.mp4`

The video will automatically be used as the background in the hero section of the homepage if it exists.

## Example
```
public/Hero-video/hero-background.mp4
```
